# Marketplace Bolo no Pote

O objetivo desse projeto é desenvolver uma aplicação Web que permita que o usuário encomende doces (bolo no pote) de forma fácil e intuitiva.

A plicação facilitará o contato entre o usuário e o administrador, assim como manter o usuário sempre atualizado sobre o pedido (encomenda).

## Instruções de utilização

### Em desenvolvimento

Para acessar a aplicação em desenvolvimento, é necessário executar o Backend e o Frontend, de maneira que ambos apontem para as URL's em desenvolvimento (localhost). Para executar o Frontend, é necessário instalar as dependências do Node.js.

### Em produção

Para acessar a aplicação em produção, acesse pela seguinte URL: https://tis-3.vercel.app/
